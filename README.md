# PaceWisp | Advanced ISP Management Hub

A high-performance management interface designed for ISPs to orchestrate PPPoE subscribers, MikroTik infrastructure, and financial operations. Developed by **[pacewisp.co.ke](https://pacewisp.co.ke)**.

## 🚀 Overview
**PaceWisp** is a premium management platform that centralizes the operation of a modern ISP. It provides professional tools for subscriber lifecycle management, bandwidth tiering, and financial reconciliation within a unified, high-fidelity UI.

## 📋 Core Capabilities

### 🌐 Subscriber Operations
*Detailed management of PPPoE secrets, active sessions, and status toggles (Enabled/Disabled). Supports dynamic package switching and expiration date adjustments.*

### 💰 Financial Intelligence
*Integrated M-Pesa C2B ledger reconciliation linked to subscriber profiles. Features an operational expense tracker (Rent, Utilities, Backhaul) to monitor net profitability in real-time.*

### 📡 Infrastructure & Packages
*Real-time health monitoring of MikroTik router nodes (CPU, Uptime, Load) and a dedicated system for bandwidth package tiering (5Mbps, 10Mbps, 20Mbps).*

### 💬 Communication Hub
*Centralized SMS dispatch center for automated expiration alerts, payment confirmations, and custom maintenance notifications to clients.*

### 👥 Team & Permissions
*Role-based access control (RBAC) to manage staff permissions for routers, billing, and customer data sensitive operations.*
